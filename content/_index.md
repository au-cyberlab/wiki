# CyberLab Wiki

The CyberLab Wiki hosts the official documentation for Adelaide University's [CyberLab][1]. It is intended for CyberLab users and contributors alike, as well as those interested in CyberLab's technical implementation.

We strive to keep the wiki as intuitive and up-to-date as possible. If you have questions or concerns, please reach out to the CyberLab tutors.

## Downloading the wiki

The entire wiki is written in Markdown, and converted to HTML using Hugo. The Markdown files (and images) are available under `content/` in the [wiki Git repository][2].

```
git clone https://github.com/au-cyberlab/wiki
cd wiki/content/
```
